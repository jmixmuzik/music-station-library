# Music Library System

NAS에 저장된 음악 파일을 관리하고 태그 기반으로 필터링/탐색하는 시스템입니다.

## 프로젝트 구조

```
music-station/
├── frontend/          # 프론트엔드 (순수 HTML/CSS/JS)
├── backend/           # 백엔드 (Flask)
├── docs/              # 기획서 문서
├── docker-compose.yml # Docker Compose 설정
└── nginx.conf         # Nginx 설정
```

## 개발 단계

- **Phase 1**: 프론트엔드 (Mock 데이터)
- **Phase 2**: 백엔드 API
- **Phase 3**: 데이터베이스 (SQLite)

## 기술 스택

- Frontend: 순수 HTML/CSS/JavaScript
- Backend: Flask (Python)
- Database: SQLite
- Deployment: Docker Compose + Nginx

## 시작하기

자세한 내용은 `docs/` 폴더의 기획서를 참고하세요.
